# JPA 소개

## JPA란?

**JPA**(Java Persistence API)란 Java ORM 기술에 대한 API 표준 명세이다.

## ORM

**ORM**이란 객체와 테이블을 매핑해서 패러다임의 불일치를 개발자가 대신 해결하는 것이다.

**패러다임의 불일치**: 자바 언어는 객체지향이고 DB는 데이터 중심으로 패러다임의 불일치가 발생
